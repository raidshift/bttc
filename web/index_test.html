a fully decentralized protocol for private transactions on BitTorrent-Chain
read more
  • Raidshift protocol is based on Tornado Cash
  • Its Smart Contracts are immutable, have no admins, and the proofs are based on strong cryptography (zk-SNARKs)
  • Neither Raidshift nor any other party collects protocol fees from deposits or withdrawals - the protocol is free of charge !
  • How does it work ?
    • A user deposits USDT together with the hash of a random private note to a Raidshift Smart Contract
    • The deposit can only be withdrawn by a user in possession of the private note
    • For optimal privacy the user should wait some time before withdrawing
  • How does Raidshift achieve privacy?
    • Raidshift improves transaction privacy by breaking the on-chain link between sender address (deposit) and receiver address (withdraw)
    • It uses Smart Contracts that only accept a fixed amount of USDT to be deposited that can then be withdrawn by a different address
    • There is no way to link the withdrawal to the deposit, ensuring absolute privacy
